# Image Viewer

Minimal web tool for pasting images from the clipboard and doing quick edits

![preview image](preview-image.png)

## Features

- Paste an image from the clipboard (Ctrl/Cmd + V) to open it.
- Draw with left mouse button. Use the vertical slider to change brush size.
- Change color with the round color picker.
- Undo / Redo with the history buttons or Ctrl+Z / Ctrl+Y.
- Pan with middle mouse (press) and drag; zoom with Ctrl + mouse wheel or `+` / `-` keys.
- Toggle fullscreen, download the edited image, or enable pixel-perfect mode using the right controls.
- View image width, height and size in the info panel.
